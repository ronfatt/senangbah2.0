# Avatar Economy

## Goal

Give students a playful reason to keep learning by turning strong work into visible personal style.

The avatar system should make progress feel:

- collectible
- personal
- expressive
- earned

## Core loop

1. Student completes a mission.
2. AI awards 1 to 3 stars based on quality.
3. Stars convert into Star Points.
4. Student spends Star Points in the Avatar Closet.
5. New outfits, shoes, and accessories make the avatar feel more unique.
6. Student comes back to earn the next item.

## Reward model

### Stars

- 1 star = decent work
- 2 stars = strong work
- 3 stars = standout work

### Star Points

- 1 star -> 80 points
- 2 stars -> 140 points
- 3 stars -> 220 points

This keeps the single-session reward clear while giving students a longer-term currency to save and spend.

## Closet structure

Initial slots:

- Hair
- Top
- Bottom
- Shoes
- Accessory

Later expansion:

- Outerwear
- Bag
- Background
- Aura / effects

## Item economy

### Starter items

Every student gets a few free basics so their avatar is usable on day one.

### Shop items

Bought with Star Points.

Suggested price bands:

- Common: 200 to 400 points
- Rare: 600 to 1000 points
- Epic: 1400 to 2200 points

### Achievement items

Unlocked by learning behavior, not just spending.

Examples:

- complete 5 English missions
- hold a 3-day streak
- reach 85% average accuracy
- finish a full weekly target

## Design direction

The first item sets should feel intentional rather than random.

Suggested themes:

- Study Core
- Sporty Pop
- Galaxy Explorer
- Exam Ace
- Neo Math

Subject-themed cosmetics can become premium motivation later:

- English scholar set
- BM klasik set
- Sejarah patriot set
- Geografi explorer set
- Math neon set

## Product rules

1. The first purchase should be easy.
2. Not every good-looking item should be premium-only.
3. Students should be able to show identity, not just status.
4. Some items should require both points and achievement milestones.
5. Closet progress should be visible on the dashboard.

## MVP scope

### Phase 1

- Avatar Closet page
- 5 equipment slots
- starter and shop items
- point balance from learning activity
- equipped item save model

### Phase 2

- achievement-locked cosmetics
- rarity labels
- subject-themed collections
- "new item unlocked" celebration

### Phase 3

- weekly limited drops
- seasonal collections
- social sharing of avatar cards

## Why this helps conversion

Students do not only pay for more subjects.
They also pay to keep momentum, collect better cosmetics, and keep building a version of the platform that feels like theirs.
